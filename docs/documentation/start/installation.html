推荐 选择 1. 使用 CDN

您可以直接从 jsDelivr 导入 CSS 文件：

{% include docs/components/tabs.html button_a="CSS @import" button_b="HTML " item_a=jsdelivr_a item_b=jsdelivr_b %}

从 v1 版本开始，主版本对于 RTL 内容也可以正常工作，特别感谢逻辑属性的使用。因此，不需要从 Bulma 分离单独的 RTL 版本出来再开发。

Bulma 也可以通过 cdnjs 获取。
