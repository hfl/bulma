Bulma 中每个元素都是移动设备优先并且做了竖直阅读优化，所以会默认使用移动设备：

  • columns 会竖向堆叠
  • level 组件会竖向显示其子内容
  • nav 菜单会隐藏

您也可以强制对 columns 或 level 通过添加 is-mobile 修饰符使用 水平（horizontal） 布局。
